# Watch Inventory Atelier Public

A mobile-first local web app for watch buy, sell, and trade record-keeping.

## Public variant

This version is intentionally app-only.

It contains:
- the app code
- the UI
- the local server

It does not contain:
- private inventory data
- uploaded images
- proprietary user records
- monthly-payment workflow

## Run locally

```bash
node server.mjs
```

Then open:

- http://127.0.0.1:4313
